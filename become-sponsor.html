Become a Festival Sponsor

We rely on the generosity of our community to make this festival possible. If you are interested in making a donation, please send us a Venmo to @portlandargentinianfestival. If you are a business looking to sponsor, here are our options:

Sponsorships Levels

Diamante ($5000)

  • EXCLUSIVE Diamond benefit: Festival Name (Sponsored by)
  • Promotional Booth
  • Social media presence (several posts)
  • On stage short company presentation (between shows)
  • Website promotional link with logo
  • Program / signage logo

Platino ($2500)

  • Platinum benefit: Presenting sponsor
  • Promotional Booth
  • Social media presence (several posts)
  • Live Stage mention
  • Website promotional link with logo
  • Program

Oro ($1000)

  • Gold benefit: Show segment sponsor
  • Promotional Booth (staff optional)
  • Social media presence (3 posts)
  • Live Stage mention
  • Website promotional link with logo
  • Program

Plata ($500)

  • Promotional table (small/no staff)
  • Social media presence (2 posts)
  • Live Stage mention
  • Website promotional link with logo
  • Program

Cobre ($200)

  • Social media presence (1 post)
  • Program

Amigo ($100)

  • Your name or company mentioned in the program and web page
